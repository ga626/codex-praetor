## 4. 外部研究的访问缺口

- OpenAI Codex 官方正文页面在当前网络出现 TLS `UNEXPECTED_EOF`/`ERR_CONNECTION_CLOSED`，保留官方搜索摘要并标注为中强，不用它支撑超出摘要范围的细节。
- GitHub、MCP、VS Code、JetBrains、Temporal、NIST 官方正文抽取成功。
- KnowledgeRadar 单个 web query 的 coverage 经常报告 `domain_diversity_low`；本报告采用跨生态、多官方来源交叉，而不是把单次搜索结果数量当证据饱和。
- 未使用登录态、账号数据库、provider token、截图或本机私密日志。
